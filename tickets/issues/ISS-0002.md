# Implement Chat Interface CLI

## Description
## Overview
Create interactive chat CLI for Commander mode.

## Requirements
- REPL-style interface with `Commander>` prompt
- Command parsing for built-in commands
- Natural language passthrough to connected instance
- Rich output formatting (colors, tables)
- History and readline support

## Built-in Commands
- `list` / `instances` - Show active instances
- `start <path> [--framework cc|mpm]` - Start new instance
- `connect <name>` - Connect to instance
- `disconnect` - Disconnect from current instance
- `stop <name>` - Stop instance
- `status` - Show current connection status
- `help` - Show commands
- `exit` / `quit` - Exit commander

## Key Files
- `src/claude_mpm/commander/chat/cli.py`
- `src/claude_mpm/commander/chat/commands.py`
- `src/claude_mpm/commander/chat/repl.py`
