# Implement Framework Selection & Instance Startup

## Description
## Overview
Support starting instances with different frameworks (cc, mpm, codex).

## Frameworks
- `cc` - Claude Code (`claude` command)
- `mpm` - Claude MPM (`claude-mpm` command with PM agent)
- `codex` - OpenAI Codex (future)

## Requirements
- Detect available frameworks on system
- Start correct command in tmux pane
- Pass project path and git context
- Return confirmation with tmux session name
- Track framework per instance

## Key Files
- `src/claude_mpm/commander/frameworks/base.py`
- `src/claude_mpm/commander/frameworks/claude_code.py`
- `src/claude_mpm/commander/frameworks/mpm.py`
- `src/claude_mpm/commander/instance_manager.py`

## Acceptance Criteria
- [ ] `start ~/project --framework cc` spawns Claude Code
- [ ] `start ~/project --framework mpm` spawns Claude MPM
- [ ] Returns tmux session name
- [ ] Reports git branch/status on startup
- [ ] Instance tracked in registry

3-4 days
