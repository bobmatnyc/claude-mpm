# Development Guide

Complete development documentation for contributing to Claude MPM.

## Quick Start

```bash
# Clone repository
git clone https://github.com/bobmatnyc/claude-mpm.git
cd claude-mpm

# Complete development setup
make dev-complete

# Run tests
make test

# Run quality checks
make quality
```
