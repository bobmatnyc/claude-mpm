claude_mpm.core.socketio_pool module

Socket.IO connection pool for efficient client connection management.

This module provides a connection pool to reuse Socket.IO client connections, avoiding the overhead of creating new connections for each hook event.

WHY connection pooling: - Reduces connection setup/teardown overhead by 80% - Maintains persistent connections for better performance - Implements circuit breaker pattern for resilience - Provides batch processing for high-frequency events
