claude_mpm.services.agents.memory.agent_persistence_service module

Agent Persistence Service (Stub)

WHY: This is a stub implementation to support the AgentLifecycleManager integration. The actual persistence is now handled by AgentManager, but we maintain this interface for backward compatibility.

DESIGN DECISION: Creating a minimal stub because: - AgentManager handles the actual file persistence - This maintains the existing API contract - Allows for future extension if needed
