Core Types

Central type definitions for Claude MPM.

This module provides shared type definitions to prevent circular import dependencies. By centralizing commonly used types, we avoid the need for cross-module imports that can create circular dependency chains.

WHY: Circular imports were causing ImportError exceptions throughout the codebase. By extracting shared types to this central location, modules can import types without creating dependency cycles.

DESIGN DECISION: Only include types that are shared across multiple modules. Module-specific types should remain in their respective modules.
