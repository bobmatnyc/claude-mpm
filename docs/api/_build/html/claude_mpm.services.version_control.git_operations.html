claude_mpm.services.version_control.git_operations module

Git Operations Manager - Core Git operation automation for Version Control Agent.

This module provides comprehensive Git operation management including: 1. Branch creation and management 2. Merge operations and conflict resolution 3. Remote operations and synchronization 4. Quality gate integration 5. Automatic branch lifecycle management
