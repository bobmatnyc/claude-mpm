claude_mpm.services.framework_claude_md_generator package

Framework INSTRUCTIONS.md Generator Service

This service provides structured generation of the framework INSTRUCTIONS.md template (legacy: CLAUDE.md) with auto-versioning, section management, and deployment capabilities.

class claude_mpm.services.framework_claude_md_generator.FrameworkClaudeMdGenerator(target_filename: str = 'INSTRUCTIONS.md')[source]

Bases: object

Generates and manages the framework INSTRUCTIONS.md template (legacy: CLAUDE.md) with structured sections, auto-versioning, and deployment capabilities.

This is the main facade class that coordinates all the submodules.
