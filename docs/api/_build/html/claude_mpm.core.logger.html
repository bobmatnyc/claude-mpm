claude_mpm.core.logger module

Unified logging system for Claude MPM.

This module consolidates features from: - utils/logger.py (simple console/file logging) - core/logging_config.py (Rich formatting, JSON, streaming) - core/project_logger.py (project-local logging, statistics)
