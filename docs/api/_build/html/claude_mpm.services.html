claude_mpm.services package

Services for Claude MPM.

This module provides backward compatibility for the reorganized service layer. Part of TSK-0046: Service Layer Architecture Reorganization

New structure: - core/: Core interfaces and base classes - agent/: Agent-related services - communication/: SocketIO and WebSocket services - project/: Project management services - infrastructure/: Logging and monitoring services
