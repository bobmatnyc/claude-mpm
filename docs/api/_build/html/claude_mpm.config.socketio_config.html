claude_mpm.config.socketio_config module

Socket.IO server configuration for different deployment scenarios.

This module provides configuration management for Socket.IO servers across different deployment environments and installation methods.

WHY configuration management: - Enables different settings for development vs production - Supports multiple deployment scenarios (local, PyPI, Docker, etc.) - Provides environment-specific defaults - Allows runtime configuration overrides
