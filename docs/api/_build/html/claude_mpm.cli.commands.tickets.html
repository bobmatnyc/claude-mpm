claude_mpm.cli.commands.tickets module

Tickets command implementation for claude-mpm.

WHY: This module provides comprehensive ticket management functionality, allowing users to create, view, update, and manage tickets through the CLI. It integrates with ai-trackdown-pytools for persistent ticket storage.

DESIGN DECISION: We implement full CRUD operations plus search and workflow management to provide a complete ticket management system within the claude-mpm CLI. The commands mirror the scripts/ticket.py interface for consistency.

Main ticket command dispatcher.

WHY: This function routes ticket subcommands to their appropriate handlers, providing a single entry point for all ticket-related operations.
