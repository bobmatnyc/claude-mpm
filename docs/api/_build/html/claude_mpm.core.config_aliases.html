claude_mpm.core.config_aliases module

Configuration alias management for Claude PM Framework.

Manages friendly directory aliases for configuration paths, allowing users to reference configurations using memorable names instead of full paths.

Example

claude-pm –config personal # Resolves to ~/.claude-mpm/configs/personal/ claude-pm –config work # Resolves to ~/work/claude-configs/

Aliases are stored in ~/.claude-mpm/config_aliases.json
