claude_mpm.utils.robust_installer module

Robust dependency installer with retry logic and fallback strategies.

WHY: Network issues and temporary unavailability can cause dependency installation to fail. This module provides resilient installation with automatic retries, fallback strategies, and clear error reporting.

DESIGN DECISION: We implement exponential backoff for retries and provide multiple installation strategies (pip, conda, source) to maximize success rate.
