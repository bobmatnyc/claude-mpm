claude_mpm.utils.paths module

Path resolution utilities for Claude MPM.

This module provides centralized path discovery and resolution logic to avoid duplication across the codebase.

class claude_mpm.utils.paths.PathResolver[source]

Bases: object

Centralized path resolution for Claude MPM.

This class consolidates all path discovery logic to avoid duplication across different modules. It handles various scenarios including: - Running from different directories - Installed vs development mode - Missing directories

All methods use caching to improve performance for repeated lookups.
