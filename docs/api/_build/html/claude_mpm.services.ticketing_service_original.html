claude_mpm.services.ticketing_service_original module

Ticketing Service

Core service that wraps ai-trackdown-pytools for simplified ticket management. Provides a clean interface for PM orchestration and agent ticket operations.

Key Features: - Singleton pattern for consistent ticket management - Simplified API wrapping ai-trackdown-pytools - Automatic ticket directory management - Thread-safe operations - Comprehensive error handling and logging - Integration with Claude PM Framework

Usage:

from claude_pm.services.ticketing_service import TicketingService

# Get singleton instance ticketing = TicketingService.get_instance()

# Create a ticket ticket = ticketing.create_ticket(

title=”Implement new feature”, description=”Detailed description”, priority=”high”

)

# List tickets tickets = ticketing.list_tickets(status=”open”)

# Update ticket ticketing.update_ticket(“CLAUDE-001”, status=”in_progress”)
