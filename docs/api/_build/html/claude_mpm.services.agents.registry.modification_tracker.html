claude_mpm.services.agents.registry.modification_tracker module

Agent Modification Tracker - Consolidated Service

Comprehensive agent modification tracking and persistence system for monitoring agent changes across the three-tier hierarchy with real-time detection and intelligent persistence management.

Key Features: - Real-time file system monitoring for agent changes - Modification history and version tracking - Agent backup and restore functionality - Modification validation and conflict detection - SharedPromptCache invalidation integration - Persistence storage in hierarchy-appropriate locations - Change classification (create, modify, delete, move)

This is a consolidated version of the agent_modification_tracker service, combining all functionality into a single module for better maintainability.
