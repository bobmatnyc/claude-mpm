claude_mpm.services.memory.router module

Memory Router Service

Routes memory commands to appropriate agents based on content analysis.

This service provides: - Content analysis to determine target agent - Support for “remember”, “memorize”, “add to memory” commands - Routing decision with reasoning - Context-aware agent selection

WHY: When users say “remember this for next time”, the system needs to determine which agent should store the information. This service analyzes content to make intelligent routing decisions for PM delegation.

DESIGN DECISION: Uses keyword matching and context analysis rather than ML models for simplicity and transparency. Routing decisions include reasoning to help users understand why content was assigned to specific agents.
