🎹 Keyboard Navigation Test Guide

✅ What's New:

  • Arrow key navigation (↑/↓) for events list
  • Auto-updating module viewer as you navigate
  • Blue highlight shows selected event
  • Smooth scrolling to keep selection in view

📝 How to Test:

  1. Open the Socket.IO Dashboard
  2. Wait for some events to appear in the list
  3. Click anywhere on the dashboard to focus it
  4. Press ↓ to select the first event
  5. Continue pressing ↑/↓ to navigate

🔍 What to Check:

  • Selected event has blue background
  • Module viewer (left panel) updates automatically
  • Arrow keys wrap at top/bottom of list
  • Selected event scrolls into view

🐛 Debugging:

Open browser console (F12) and look for:

  • Arrow navigation: 1 from X to Y
  • Selected event: Y [event type]
  • updateModuleViewer called, eventsByClass size: X
  • updateEventsByClass called with: [event object]

Quick Test:

Expected Console Output:

Arrow navigation: 1 from -1 to 0
updateEventsByClass called with: {type: "session.start", ...}
Event class: session
updateModuleViewer called, eventsByClass size: 3
Selected event: 0 session.start
